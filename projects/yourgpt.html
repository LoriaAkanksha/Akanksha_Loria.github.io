Project Overview

YourGPT is an advanced AI system that integrates Large Language Models (LLMs), vector databases, and custom user workflows. It allows for:

  • Real-time Q&A retrieval
  • Document-based knowledge ingestion
  • Multi-agent task delegation

Key Technologies

  • Python (FastAPI, Streamlit)
  • Azure OpenAI / OpenAI GPT Models
  • Langchain for context-based retrieval
  • Vector DB (ChromaDB) for embeddings
  • Multi-agent orchestration (CrewAI / custom frameworks)
